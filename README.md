# Collaborative Software Development 

Learn how to work effectively in software development teams and adopt best practices for collaboration. This advanced workshop introduces structured Git workflows, clear documentation, and automated testing and deployment through continuous integration (CI). You’ll practice these skills in realistic team scenarios, preparing you to contribute confidently to research software projects that meet institutional and community standards for quality and reproducibility.
